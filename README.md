# Workout-Streak-Bot - приложение для тренеровок, которое поможет в составлении своих тренировок и в поддержании темпа в них

## Описание проекта

Приложение позволяет каждому создать свой план тренеровки, или выбрать уже составленную свою тренеровку

- Пользователь выбирает или создаёт свой план тренировки
    - Пользователь загружает картинку, gif, название упражнения, название тренеровки, сколько повторений за раз выполнить, сколько подходов за тренеровку.

- У пользователя есть меню настройки своих тренеровок, редактирование картинки, названия, повторений

- Если в течении тренеровки - вся норма повторений была выполнена идеальна, то предлагается увеличить количество повторений у каждого упражнения

- WinStreak: система стрика будет выполнена следующим образом: в течении недели стрик будет 4, если за наделю не было выполнено 4 тренеровки, то стрик сбрасывается. То есть за 2 недели - максимальный стрик - 8
