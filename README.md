# Workout-Streak-Bot - бот для тренеровок
Бот, который поможет в составлении своих тренировок и в поддержании темпа в них

## Описание проекта

Бот позволяет каждому пользователю добавить в себя свой план тренеровки, или выбрать уже готовый - одобренный разработчиками.

- Пользователь выбирает или создаёт свой план тренеровки
    - Пользователь загружает картинку, gif. Также название упражнения, название тренеровки, сколько повторений за раз выполнить, сколько подходов за тренеровку.

- У пользователя есть меню настройки своих или загруженных своих тренеровок, редактирование картинки, названия, повторений

- Если в течении тренеровки - вся норма повторений была выполнена идеальна, то предлагается увеличить количество повторений у каждого упражнения

- WinStreak: система стрика будет выполнена следующим образом: в течении недели стрик будет 4, если за наделю не было выполнено 4 тренеровки, то стрик сбрасывается. То есть за 2 недели - максимальный стрик - 8

## Команды

- `/start` - запустить бота
